## Convention

Every audit-fix commit MUST carry the trailer:

```
Closes audit <ID>
```

Where `<ID>` is the audit code from `docs/HALF_BAKED.md` or `docs/planning-003.md` (e.g. `S-1`, `K-2`, `A.1.3`, `WT 43`). Multiple closures: comma-separated.
